# 🎬 AI Video Assistant with RAG

This is a project inspired by **Sheryians AI School's** Generative AI and Agentic AI tutorial series. It implements a complete local Retrieval-Augmented Generation (RAG) system to chat with video content transcripts.

## 🚀 How to Run locally

1. Navigate to the project directory:
   ```bash
   cd "Generative AI/AI_Video_Assistant_RAG"
   ```

2. Install dependencies:
   ```bash
   pip install streamlit langchain langchain-community faiss-cpu
   ```

3. Start the Streamlit application:
   ```bash
   streamlit run app.py
   ```
