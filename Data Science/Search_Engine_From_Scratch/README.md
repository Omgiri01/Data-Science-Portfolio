# 🔍 TF-IDF Search Engine from Scratch

This project implements a text-indexing and search retrieval system from scratch in pure Python. Inspired by systems components in **codecrafters-io/build-your-own-x**, it demonstrates document tokenization, building an inverted index, calculating Term Frequency-Inverse Document Frequency (TF-IDF) weights, and vector space similarity ranking (Cosine Similarity).

## 📂 Key Components
- `engine.py`: Implements tokenization, inverted index, TF/IDF weight logs, and cosine similarity queries.
- `demo.py`: Indexes a local text corpus and returns ranked results for sample queries.

## 🚀 Setup & Execution
1. Run the demonstration search script:
   ```bash
   python demo.py
   ```
