## 📂 Project Showcase

### 1. [CFD_Surrogate_Modeling](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/CFD_Surrogate_Modeling)
- **Objective**: Construct deep learning neural networks to act as surrogate estimators for expensive fluid flow simulations.
- **Key Features**: Ingests OpenFOAM mesh data, trains convolutional neural models on velocity/pressure grid parameters, and estimates fluid fields instantly without running slow solvers.
- **Tech Stack**: PyTorch, OpenFOAM, NumPy, Matplotlib.

### 2. [Predictive_Maintenance_IoT](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/Predictive_Maintenance_IoT)
- **Objective**: Develop systems that predict machine failures before they happen using IoT sensor data (vibration, pressure, temperature).
- **Key Features**: Processes C-MAPSS aircraft engine datasets, cleans multi-sensor readings, and trains predictive classifiers estimating Remaining Useful Life (RUL).
- **Tech Stack**: Scikit-Learn, Pandas, NumPy, Time Series analysis.

### 3. [financial_analysis_python](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/financial_analysis_python)
- **Objective**: Conduct extensive financial market research and analysis utilizing Python, Pandas, and NumPy.
- **Key Features**: Implements portfolio optimization models (Markowitz Mean-Variance framework), calculates Sharpe ratios, performs stock asset tracking, and builds risk-return indicators.
- **Tech Stack**: Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebooks.

### 4. [Search_Engine_From_Scratch](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/Search_Engine_From_Scratch)
- **Objective**: Build a custom text indexing and document retrieval search engine.
- **Key Features**: Implements tokenization, inverted vocabulary mapping, term frequency log scaling, and TF-IDF cosine-similarity query matching.
- **Tech Stack**: Python, Mathematics.

### 5. [Real-Time Financial Fraud Detection Pipeline](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/README.md#project-1-real-time-financial-fraud-detection-pipeline)

- **Tech Stack**: XGBoost, LightGBM, SHAP, Python.

### 6. [Production-Grade Customer Churn & Drift Analytics](file:///c:/Users/OM%20GIRI/Downloads/DS%20Projects/Data%20Science/README.md#project-2-production-grade-customer-churn--drift-analytics)
